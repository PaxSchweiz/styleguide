# pax-carousel

The `pax-carousel` is a multi-item container. The items are provided via nested `pax-carousel-item` components.

## Notes

* Multiple independent carousels can be added to a single page.
* Depends on <pax-link href="https://swiperjs.com/">Swiper</pax-link> (MIT license).

## Basic

<pax-carousel>
    <pax-carousel-item>
        <img src="http://placehold.it/1024x400" alt="Slide 1"/>
    </pax-carousel-item>
    <pax-carousel-item>
        <img src="http://placehold.it/1024x400" alt="Slide 2"/>
    </pax-carousel-item>
    <pax-carousel-item>
        <img src="http://placehold.it/1024x400" alt="Slide 3"/>
    </pax-carousel-item>
</pax-carousel>
