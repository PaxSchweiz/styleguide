# pax-headline

Headline is a wrapper element for headlines from level 1 to 6.

* Without a `level` attribute a `<h2>` headline is rendered.


## Basic

<pax-headline>Pax Headline</pax-headline>

```html
<pax-headline>Pax Headline</pax-headline>
```
