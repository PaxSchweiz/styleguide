Usage of Pax components NPM content delivery

Access components via NPM URL without installation - accessible within Pax and Internet. This is using unpkg.com. The main URL for web-components is: https://unpkg.com/@devatpax/web-components@latest/dist/pax. Replace the version "latest" with the version needed (like "2.1.0") or leave it to get the latest. 

<!-- BEGIN_WEBCOMPONENTS include web components -->
<link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Source+Sans+Pro:ital,wght@0,400;1,900&display=swap">
<script type="module" src="https://unpkg.com/@devatpax/web-components@latest/dist/pax/pax.esm.js"></script>
<script nomodule src="https://unpkg.com/@devatpax/web-components@latest/dist/pax/pax.js"></script>
<link rel="stylesheet" type="text/css" href="https://unpkg.com/@devatpax/web-components@latest/dist/pax/pax.css">
<!-- END_OF_WEBCOMPONENTS -->

Use with local installation using npm. Add the npm package to your own project using npm i --save @devatpax/web-components. The "--save" adds the dependency to your existing nodejs project (only if the file package.json is available) 

<!-- BEGIN_WEBCOMPONENTS include web components -->
<link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Source+Sans+Pro:ital,wght@0,400;1,900&display=swap">
<script type="module" src="node_modules/@devatpax/web-components/pax.esm.js"></script>
<script nomodule src="node_modules/@devatpax/web-components/pax.js"></script>
<link rel="stylesheet" type="text/css" href="node_modules/@devatpax/web-components/pax.css">
<!-- END_OF_WEBCOMPONENTS -->
