<pax-title>Gebrauch der Pax Komponenten</pax-title>
<pax-headline level="3">Nutzung für eine simple HTML Seite</pax-headline>
Die einfachste Nutzung erfolgt über ein Content Delivery Network (CDN) und benötigt keine lokale Installation.
Für diese Komponenten steht folgendes CDN zu Verfügung: <code>https://unpkg.com/@devatpax/web-components@latest/dist/pax</code>. Replace
Mit "latest" ist die aktuellste Version referenziert. Für eine simple HTML Seite kann folgender HTML code im "head" eingefügt werden
und danach können die Komponenten angewandt werden.

```
<!-- BEGIN_WEBCOMPONENTS include web components -->
<link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Source+Sans+Pro:ital,wght@0,400;1,900&display=swap">
<script type="module" src="https://unpkg.com/@devatpax/web-components@latest/dist/pax/pax.esm.js"></script>
<script nomodule src="https://unpkg.com/@devatpax/web-components@latest/dist/pax/pax.js"></script>
<link rel="stylesheet" type="text/css" href="https://unpkg.com/@devatpax/web-components@latest/dist/pax/pax.css">
<!-- END_OF_WEBCOMPONENTS -->
```

Für Entwicklungen können die Komponenten auch via dem Node Package Manager (NPM) lokal installiert werden und anstelle
über das CDN lokal genutzt werden. Die Installation erfolgt über: <code>npm i --save @devatpax/web-components</code>.

```
<!-- BEGIN_WEBCOMPONENTS include web components -->
<link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Source+Sans+Pro:ital,wght@0,400;1,900&display=swap">
<script type="module" src="node_modules/@devatpax/web-components/pax.esm.js"></script>
<script nomodule src="node_modules/@devatpax/web-components/pax.js"></script>
<link rel="stylesheet" type="text/css" href="node_modules/@devatpax/web-components/pax.css">
<!-- END_OF_WEBCOMPONENTS -->
```
