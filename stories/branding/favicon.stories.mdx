# Favicon

Ein Favicon ist ein kleines, 16×16 oder 32×32 Pixel messendes Icon, Symbol oder Logo, das von Webbrowsern verwendet
wird, um eine Website auf wiedererkennbare Weise zu kennzeichnen.
